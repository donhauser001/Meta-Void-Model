## 数据流向 (Data Flow)

```
[Potentiality Map]  ──θ──►  [Tension Query]  ──ω──►  [Convolution]  ──►  [Snapshot]
     (静态)                    (路径选择)            (频谱渲染)          (离散输出)
```

**关键转换点：**

1. **θ 路径锁定**：从无限潜能中选择特定区域
2. **ω 频谱设定**：决定该区域的渲染精度
3. **⊗ 卷积运算**：非线性张力扰动
4. **快照输出**：一帧离散的现实单元
