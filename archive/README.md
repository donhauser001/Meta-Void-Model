## 为什么保留这些文件？

1. **溯源比对**：模块化拆分可能存在逻辑微调或文字删减，保留原稿方便产生歧义时进行对照
2. **离线阅读**：部分读者习惯下载完整文档一次性阅读
3. **历史见证**：记录 MVM 从"一部著作"进化为"一个系统"的起点
