# 贡献指南 (Contributing to MVM)

[![English](https://img.shields.io/badge/English-Version-blue)](../en/CONTRIBUTING.md)

感谢你关注 **Meta-Void-Model (MVM)**。

本仓库不仅仅是一个理论的陈列馆，更是一个开放的**逻辑压力测试场**。由于本模型试图在物理学、信息论与现象学之间建立跨学科联系，我深知单一个体的视角必然存在局限。

我邀请具有理性和严谨态度的探索者通过以下方式参与本模型的进化。

---

## 01. 参与准则 (Principles)

在参与贡献之前，请确保你认同以下原则：

| 准则 | 说明 |
|------|------|
| **逻辑优先** | 我欢迎对理论的任何质疑，但请基于逻辑推导而非直觉感悟 |
| **术语对齐** | 在讨论时，请尽量使用本项目 Glossary/术语表（待建）中定义的含义，或明确给出你的定义 |
| **非玄学化** | 虽然探讨意识，但我拒绝缺乏逻辑支撑的超自然描述。请保持学术性的克制 |

---

## 02. 我寻找的贡献 (Areas of Focus)

目前重点关注以下几个方向的突破：

### A. 逻辑证伪 (Logical Falsification)

如果你发现理论框架中存在**"逻辑断裂点"**（Logical Breaks）或自相矛盾之处，请提交 Issue。

> **这是对本项目最高级别的贡献。**

### B. 数学形式化 (Mathematical Formalization)

目前 $\theta$（意识路径）与 $\omega$（频谱）的交互仍处于概念描述阶段。我寻找能够利用以下工具对其进行严密数学描述的贡献：

- **范畴论 (Category Theory)**：描述从非存在场到显现态的态射
- **拓扑数据分析 (TDA)**：研究意识路径在元虚空张力结构中的拓扑性质
- **信息论**：量化快照渲染过程中的压缩与信息损耗

### C. 物理学与神经科学映射 (Interdisciplinary Mapping)

寻找 MVM 模型与现有实验证据的关联，例如：

- 量子力学中的**延迟选择实验**在 MVM 中的解释路径
- 神经科学中关于"受限感知"与 MVM"采样机制"的类比

---

## 03. 参与流程 (Contribution Process)

### 提交 Issue

| 标签 | 用途 |
|------|------|
| `[Refutation]` 🔴 | **质疑类**：清晰描述逻辑矛盾点 |
| `[Suggestion]` 🔵 | **建议类**：说明该建议如何增强模型的解释力 |
| `[Inquiry]` 🟢 | **提问类**：针对不清晰的概念寻求解释 |

### 提交 Pull Request (PR)

- **文档修订**：如果你发现了排版错误或表述模糊点，欢迎直接 PR
- **新增章节**：如果你想为 `Appendix/` 或 `Theory/` 贡献新的视角，请先通过 Issue 讨论其逻辑一致性
- **代码/符号化**：欢迎提交关于模型参数化的代码实现（Python / Mathematica 等）

---

## 04. 讨论渠道 (Discussions)

对于非结构化的、发散性的思考，请使用 GitHub 的 **Discussions** 板块。

设置了以下分类：

| 分类 | 主题 |
|------|------|
| `#The-Void` | 关于"非存在"本质的哲学探讨 |
| `#Rendering-Logic` | 关于快照与离散现实的物理/数学讨论 |
| `#Experience-Echo` | 关于第一人称意识体验的案例分析 |
