# Symbols / 符号与公式

本目录存放 MVM 模型的形式化符号定义与数学推导。

## 核心符号

| 符号 | 含义 |
|:----:|------|
| $\omega$ | Spectrum（频谱）：显现的质感与解析度 |
| $\theta$ | Path（路径）：意识接入元虚空的特定路径 |
| $\otimes$ | 非线性张力卷积算子 |

## 规划内容

- `core-definitions.md` — 核心符号的严格定义
- `derivations/` — 数学推导过程
- `formalization.md` — 形式化尝试
